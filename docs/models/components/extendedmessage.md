# ExtendedMessage

Extended Agent-to-Agent protocol message with support for tool calls and tool results. Extends the base A2A message format with Orquesta-specific features.

## Example Usage

```typescript
import { ExtendedMessage } from "@orq-ai/node/models/components";

let value: ExtendedMessage = {
  kind: "message",
  messageId: "<id>",
  role: "user",
  parts: [],
};
```
