# Metadata

Request metadata forwarded for dynamic-budget matching
 (`metadata.team == "ml"`). Free-form JSON object from the request
 body's `metadata` field.

## Example Usage

```typescript
import { Metadata } from "@orq-ai/node/models/components";

let value: Metadata = {};
```
