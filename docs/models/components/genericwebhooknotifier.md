# GenericWebhookNotifier

`webhook_url` is present for generic webhook notifiers; `headers` is optional.

## Example Usage

```typescript
import { GenericWebhookNotifier } from "@orq-ai/node/models/components";

let value: GenericWebhookNotifier = {
  type: "NOTIFIER_TYPE_SLACK_WEBHOOK",
  webhookUrl: "https://infamous-soybean.net",
  id: "<id>",
  displayName: "Luz66",
  createdAt: new Date("2025-08-29T23:51:48.411Z"),
  updatedAt: new Date("2024-04-16T04:52:09.184Z"),
};
```
