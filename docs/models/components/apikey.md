# ApiKey

ApiKey is the canonical record stored in MongoDB `auth.apiKeys`.
 It is the source of truth for scope, permissions, owner, budget,
 expiration, and revocation (see ADR 0001).

## Example Usage

```typescript
import { ApiKey } from "@orq-ai/node/models/components";

let value: ApiKey = {
  apiKeyId: "<id>",
  name: "<value>",
  owner: {},
  projectScope: {},
  permissionMode: "PERMISSION_MODE_RESTRICTED",
  tokenPrefix: "<value>",
  status: "API_KEY_STATUS_UNSPECIFIED",
  createdAt: new Date("2024-07-09T19:24:40.889Z"),
  updatedAt: new Date("2024-08-17T17:42:27.645Z"),
};
```
