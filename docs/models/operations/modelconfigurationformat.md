# ModelConfigurationFormat

Specifies the output audio format. Must be one of wav, mp3, flac, opus, or pcm16.

## Example Usage

```typescript
import { ModelConfigurationFormat } from "@orq-ai/node/models/operations";

let value: ModelConfigurationFormat = "pcm16";
```

## Values

```typescript
"wav" | "mp3" | "flac" | "opus" | "pcm16"
```
