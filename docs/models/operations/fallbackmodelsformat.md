# FallbackModelsFormat

Specifies the output audio format. Must be one of wav, mp3, flac, opus, or pcm16.

## Example Usage

```typescript
import { FallbackModelsFormat } from "@orq-ai/node/models/operations";

let value: FallbackModelsFormat = "flac";
```

## Values

```typescript
"wav" | "mp3" | "flac" | "opus" | "pcm16"
```
