# ModelConfiguration2



Model configuration with parameters and retry settings.

## Example Usage

```typescript
import { ModelConfiguration2 } from "@orq-ai/node/models/operations";

let value: ModelConfiguration2 = {
  id: "<id>",
  retry: {
    onCodes: [
      429,
      500,
      502,
      503,
      504,
    ],
  },
};
```
