# RequestBodyFunctionTool

A custom function tool that allows the model to call predefined functions with structured parameters.

## Example Usage

```typescript
import { RequestBodyFunctionTool } from "@orq-ai/node/models/operations";

let value: RequestBodyFunctionTool = {
  path: "Default",
  key: "<key>",
  description: "surprisingly management aw character wretched",
  type: "function",
  function: {
    name: "<value>",
  },
};
```
