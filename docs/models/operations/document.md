# Document

Document to run OCR on. Can be a DocumentURLChunk or ImageURLChunk.


## Supported Types

### `operations.Document1`

```typescript
const value: operations.Document1 = {
  type: "document_url",
  documentUrl: "https://electric-teammate.name",
};
```

### `operations.Document2`

```typescript
const value: operations.Document2 = {
  type: "image_url",
  imageUrl: "https://digital-nightlife.name",
};
```
