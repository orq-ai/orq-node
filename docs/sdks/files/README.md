# Files

## Overview

### Available Operations

* [list](#list) - List all files
* [create](#create) - Create file
* [getContent](#getcontent) - Download file content
* [delete](#delete) - Delete file
* [get](#get) - Retrieve a file
* [update](#update) - Update file

## list

Returns a list of the files that your account has access to. orq.ai sorts and returns the files by their creation dates, placing the most recently created files at the top.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="FileList" method="get" path="/v2/files" -->
```typescript
import { Orq } from "@orq-ai/node";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.files.list({});

  console.log(result);
}

run();
```

## create

Files are used to upload documents that can be used with features like Deployments.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="FileUpload" method="post" path="/v2/files" -->
```typescript
import { Orq } from "@orq-ai/node";
import { openAsBlob } from "node:fs";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.files.create({
    file: await openAsBlob("example.file"),
  });

  console.log(result);
}

run();
```
