# Budgets

## Overview

### Available Operations

* [list](#list) - List budget configurations
* [create](#create) - Create budget configuration
* [get](#get) - Get budget configuration
* [update](#update) - Update budget configuration
* [delete](#delete) - Delete budget configuration

## list

Retrieves a paginated list of budget configurations in your workspace. Supports filtering by type (contact).

### Example Usage

<!-- UsageSnippet language="typescript" operationID="ListBudgets" method="get" path="/v2/budgets" -->
```typescript
import { Orq } from "@orq-ai/node";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.budgets.list({
    type: "contact",
    entityId: "contact_01ARZ3NDEKTSV4RRFFQ69G5FAV",
    isActive: true,
  });

  console.log(result);
}

run();
```

## create

Create a new budget configuration for an API key or contact. For API keys, provide the API key value (e.g., sk_live_xxx) in entity_id field. For contacts, provide the external ID in entity_id field.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="CreateBudget" method="post" path="/v2/budgets" -->
```typescript
import { Orq } from "@orq-ai/node";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.budgets.create({
    type: "contact",
    entityId: "user_123",
    period: "monthly",
    amount: 250,
  });

  console.log(result);
}

run();
```
