# Router.Responses

## Overview

### Available Operations

* [create](#create) - Create response

## create

Creates a model response for the given input.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createResponse" method="post" path="/v2/router/responses" -->
```typescript
import { Orq } from "@orq-ai/node";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.router.responses.create({
    model: "Taurus",
    input: [],
  });

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { OrqCore } from "@orq-ai/node/core.js";
import { routerResponsesCreate } from "@orq-ai/node/funcs/routerResponsesCreate.js";

// Use `OrqCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const orq = new OrqCore({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const res = await routerResponsesCreate(orq, {
    model: "Taurus",
    input: [],
  });
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("routerResponsesCreate failed:", res.error);
  }
}

run();
```
