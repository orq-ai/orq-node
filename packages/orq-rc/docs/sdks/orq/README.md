## getV2ModelsLitellmModels

Fetches all available models from the configured LiteLLM instance

### Example Usage

```typescript
import { Orq } from "@orq-ai/node";

const orq = new Orq({
  apiKey: process.env["ORQ_API_KEY"] ?? "",
});

async function run() {
  const result = await orq.getV2ModelsLitellmModels();

  console.log(result);
}

run();
```
