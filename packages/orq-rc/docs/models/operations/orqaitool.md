# OrqAiTool

An orq.ai platform tool reference. For MCP tools, prefer type 'mcp' with 'key' instead of 'orq:mcp' with 'tool_id'.

## Example Usage

```typescript
import { OrqAiTool } from "@orq-ai/node/models/operations";

let value: OrqAiTool = {
  type: "orq:function",
};
```
