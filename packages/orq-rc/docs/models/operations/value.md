# Value

The feedback value. For single selection of multiple choice, the value should be an array of strings. For `correction`, the value should be a string.


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `string[]`

```typescript
const value: string[] = [
  "<value 1>",
  "<value 2>",
];
```
