# ModelFormat

Specifies the output audio format. Must be one of wav, mp3, flac, opus, or pcm16.

## Example Usage

```typescript
import { ModelFormat } from "@orq-ai/node/models/operations";

let value: ModelFormat = "mp3";
```

## Values

```typescript
"wav" | "mp3" | "flac" | "opus" | "pcm16"
```
