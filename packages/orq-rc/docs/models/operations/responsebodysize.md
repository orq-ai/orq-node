# ResponseBodySize

The size of the generated images. Must be one of `256x256`, `512x512`, or `1024x1024`.

## Example Usage

```typescript
import { ResponseBodySize } from "@orq-ai/node/models/operations";

let value: ResponseBodySize = "1024x1024";
```

## Values

```typescript
"256x256" | "512x512" | "1024x1024"
```
