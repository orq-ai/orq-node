# MCPToolRun

MCP tool with inline definition for on-the-fly creation in run endpoint

## Example Usage

```typescript
import { MCPToolRun } from "@orq-ai/node/models/operations";

let value: MCPToolRun = {
  type: "mcp",
  key: "<key>",
  description:
    "whenever immaculate atop regal clamor excepting brightly coal gulp",
  mcp: {
    serverUrl: "https://sleepy-version.biz/",
    tools: [],
    connectionType: "http",
  },
};
```
