# ChoicesSystemMessage

Developer-provided instructions that the model should follow, regardless of messages sent by the user.

## Example Usage

```typescript
import { ChoicesSystemMessage } from "@orq-ai/node/models/operations";

let value: ChoicesSystemMessage = {
  role: "system",
  content: [
    {
      type: "text",
      text: "<value>",
    },
  ],
};
```

## Fields

| Field                                                                                                                        | Type                                                                                                                         | Required                                                                                                                     | Description                                                                                                                  |
| ---------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `role`                                                                                                                       | *"system"*                                                                                                                   | :heavy_check_mark:                                                                                                           | The role of the messages author, in this case `system`.                                                                      |
| `content`                                                                                                                    | *operations.ChoicesContent*                                                                                                  | :heavy_check_mark:                                                                                                           | The contents of the system message.                                                                                          |
