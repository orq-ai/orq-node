# CodeToolInput

Executes code snippets in a sandboxed environment. Must reference a pre-created code tool by key or id.

## Example Usage

```typescript
import { CodeToolInput } from "@orq-ai/node/models/components";

let value: CodeToolInput = {
  type: "code",
};
```
