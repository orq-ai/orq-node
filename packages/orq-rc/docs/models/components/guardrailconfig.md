# GuardrailConfig


## Supported Types

### `components.Boolean`

```typescript
const value: components.Boolean = {
  type: "boolean",
  value: false,
};
```

### `components.Categorical`

```typescript
const value: components.Categorical = {
  type: "categorical",
  values: [
    "<value 1>",
  ],
};
```

### `components.NumberT`

```typescript
const value: components.NumberT = {
  type: "number",
  value: 3883.98,
  operator: "ne",
};
```
