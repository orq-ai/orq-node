# Messages

## Example Usage

```typescript
import { Messages } from "@orq-ai/node/models/components";

let value: Messages = {
  role: "assistant",
  content: [
    {
      type: "image_url",
      imageUrl: {
        url: "https://svelte-plumber.org",
      },
    },
  ],
};
```
