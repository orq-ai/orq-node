# Content

The contents of the user message.


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `components.Two[]`

```typescript
const value: components.Two[] = [
  {
    type: "input_audio",
    inputAudio: {
      data: "<value>",
      format: "wav",
    },
  },
];
```
