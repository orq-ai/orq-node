# RefusalPartSchema

A message part containing a refusal message.

## Example Usage

```typescript
import { RefusalPartSchema } from "@orq-ai/node/models/components";

let value: RefusalPartSchema = {
  type: "refusal",
};
```

## Fields

| Field                                           | Type                                            | Required                                        | Description                                     |
| ----------------------------------------------- | ----------------------------------------------- | ----------------------------------------------- | ----------------------------------------------- |
| `type`                                          | *"refusal"*                                     | :heavy_check_mark:                              | The type of the content part. Always `refusal`. |
| `refusal`                                       | *string*                                        | :heavy_minus_sign:                              | The refusal message generated by the model.     |
